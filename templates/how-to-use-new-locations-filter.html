What is the New Locations Filter?

Fresh Inventory Discovery

The New Locations filter shows you only locations that have been added to TamerMap in the last 30 days. This includes:

  • New Kiosks: Fresh kiosks added to existing stores
  • New Kiosk Locations: Brand new stores that now have kiosks
  • New Retail Stores: Recently added retail locations
  • New Indie Shops: Newly discovered independent card shops

Why It's Valuable

  • First-to-Find Advantage: Discover fresh inventory before other hunters
  • Reduced Competition: New locations are less crowded initially
  • Fresh Inventory: New locations often have untouched card stock
  • Competitive Edge: Stay ahead of the hunting community
  • Success Rate Boost: Higher chances of finding valuable cards

How to Access the Filter

Step-by-Step Guide

  1. Ensure Pro Access: Make sure you have an active TamerMap Pro subscription
  2. Navigate to Map: Go to the main maps page
  3. Open Filter Panel: Click on the filter icon or panel
  4. Find New Locations: Look for "All New Locations" checkbox
  5. Enable Filter: Check the box to activate the filter
  6. View Results: See only new locations on your map
Filter Active

When the filter is active, you'll see only locations added in the last 30 days. All other locations will be hidden from view.

Understanding Filter Results

What You'll See

When the New Locations filter is active, your map will show only locations that meet these criteria:

Included Locations
  • Kiosks added in last 30 days
  • New retail store locations
  • Recently discovered indie shops
  • New kiosk installations
Hidden Locations
  • Older kiosk locations
  • Established retail stores
  • Long-known indie shops
  • Pre-existing locations
Filter Behavior

The filter automatically updates as new locations are added to TamerMap. Check back regularly to discover the latest additions!
