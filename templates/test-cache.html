Tamermap Cache Testing

This page tests the caching functionality of the dataService module. It verifies that:

  • Data is fetched from the server on first request
  • Subsequent requests use the memory cache
  • The cache can be forcibly refreshed
  • localStorage is used as a fallback when memory cache is cleared
  • Different data types (retailers, events) are cached independently
